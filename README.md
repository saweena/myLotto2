# MyLotto Compose AI Migration

A case study of modernizing a legacy Android XML application into Jetpack Compose using an AI-first development workflow.

## Goal

Migrate a personal legacy Android application to:

- Kotlin
- Jetpack Compose
- Material3

## Version 0.1 Scope

- Home Screen
- Result Checking Screen
- My Lottery Screen
